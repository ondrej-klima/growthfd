growthfd
=====
  
Fitting of FPCA-based growth curve model using Levenberg-Marquardt optimization

## Installation

```r
# Install devtools from CRAN
install.packages("devtools")
library("devtools")

# Install growthfd from GitHub:
install_github("ondrej-klima/growthfd")
```
